# To-Do Checklist App

A pastel-themed interactive to-do checklist app built using HTML, CSS, and JavaScript.

## Features
- Add and delete tasks
- Interactive checkbox completion
- Hover animations and glowing UI
- Date and time support
- Responsive aesthetic design

## Tech Stack
- HTML
- CSS
- JavaScript
